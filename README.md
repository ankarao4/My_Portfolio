# Golla Ankarao — Portfolio

A professional **3D Cybersecurity Portfolio** built with HTML5, CSS3, JavaScript, and Three.js.

## 🚀 Live Site
Deployed on **Cloudflare Pages**.

## 🛠 Tech Stack
- HTML5 / CSS3 / Vanilla JavaScript
- Three.js (3D animated background)
- Font Awesome icons
- Google Fonts (Orbitron, Inter, Fira Code)

## 📂 Structure
```
my_portfolio/
├── index.html          # Main single-page document
├── css/style.css       # All styles
├── js/
│   ├── three-bg.js     # Three.js 3D background
│   └── main.js         # Interactivity
├── assets/images/      # Images
├── resume/resume.pdf   # Resume (add your PDF here)
├── _headers            # Cloudflare cache/security headers
└── wrangler.toml       # Cloudflare Pages config
```

## 🌐 Deploy to Cloudflare Pages
```bash
# Install Wrangler if needed
npm install -g wrangler

# Login to Cloudflare
npx wrangler login

# Deploy
npx wrangler pages deploy . --project-name=golla-ankarao-portfolio
```
